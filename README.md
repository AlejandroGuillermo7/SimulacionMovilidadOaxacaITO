# SimulacionMovilidadOaxacaITO
Simulación de la movilidad vehicular en Oaxaca de Juárez, enfocada en el crucero del Instituto Tecnológico de Oaxaca. El proyecto analiza tiempos de espera, semaforización y flujo vehicular mediante Promodel, integrando reporte técnico en PDF, diagramas UML y documentación para evaluación académica.
