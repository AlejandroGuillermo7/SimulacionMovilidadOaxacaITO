# SimulacionMovilidadOaxacaITO

**Materia:** Simulación  
**Maestra:** Martínez Nieto Adelina  
**Semestre:** 5°  
**Alumnos:** Álvarez Martínez Alessandro Santiago, Enríquez Rodríguez Alejandro Guillermo y Ruíz Osante Angel Yiban  

Simulación de la movilidad vehicular en Oaxaca de Juárez, enfocada en el crucero del Instituto Tecnológico de Oaxaca. El proyecto analiza tiempos de espera, semaforización y flujo vehicular mediante Promodel, integrando reporte técnico en PDF y documentación para evaluación académica.  


## PDFs de Referencia

H. E. Gómez, ``Desarrollo de un modelo de simulación vehicular para la sincronización de semáforos,'' INAOE, Puebla, México, Informe Técnico (tesis/trabajo), 2011. [En línea]. Disponible: https://inaoe.repositorioinstitucional.mx/jspui/bitstream/1009/379/1/GomezHE.pdf
